## rails-chat-app

This is a small chat app using Rails 5 new feature - ActionCable. It has chatroom and new message notification.

**Demo**: <http://rails-example-chat-app.herokuapp.com>

## Dev-enviroment

* Ruby on Rails 5.0.x
* Ruby 2.3.3
* Puma
* Redis
* Postgresql 9.6.x
